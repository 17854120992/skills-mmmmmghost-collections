# Agent Routing Rules

## Web Research Delegation (Required)

- For tasks needing internet search, latest/current info, external citations, or multi-source verification, route to a dedicated research subagent.
- The research subagent must return only:
  1. Answer
  2. Key Findings (3-7 bullets)
  3. Evidence (claim + source URL + date)
  4. Assumptions and Gaps
  5. Confidence (high/medium/low)
- Do not paste raw search dumps, full-page captures, or long unfiltered notes into the main context.
- If confidence is low, do one targeted follow-up query, not broad re-search.

## Source Quality

- Prefer primary sources first.
- Label secondary sources explicitly.
- If sources conflict, include both and indicate which one is more reliable.
